Harmonic Densenet. Is MAC only metrics to measure the network complexity?

Key Ideas

This paper proposes new method to measure the complexity of neural network. Till now, the most common method to measure the network complexity is MACs (number of multiply-accumulate operations.), but the authors argue that this method can not be accurate in predicting inference time. They say that, when inferring over higher resolution, memory traffic can be dominating over MACs.

Following this metric, they propose a new architecture HarDNet (Harmonic Densenet) which reduce the inference time giving similar or better performance than the SOTA architectures.

What did the Authors tried to accomplish?

  • Introduction of new metrics which can be more accurate than MACs when inferring on higher resolution.
  • Proposal of new architecture HarDNet build upon the idea of Densenet which reduce the inference time by a lot.
  • Avoiding the use of 1x1 convolution as it increases the MoC.
  • Detailed experiments and analysis of architecture w.r.t. classification, object detection and segmentation tasks.
